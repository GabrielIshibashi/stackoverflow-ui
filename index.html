Your not calling the class, substitute the value for center, and use this code snippet:

                        
.center{
    text-align:center;
    align-items:center;
    align-self:center;
    align-content:center;
    justify-items:center;
    justify-self:center;
    justify-content:center;
    place-items:center;
    vertical-align:center;
    line-height: 100%;
    margin:auto;
    position:absolute;
    left:50%;
    right:50%;
    transform:translate:(-50%, -%50);
}
